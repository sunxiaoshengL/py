# 悬浮窗口连点器 (Floating Clicker)

一个功能强大、防检测的Windows悬浮窗口连点器，支持鼠标位置跟随、自定义点击间隔、紧急停止等功能。

## 🎯 功能特点

### 核心功能
- ✅ **悬浮置顶窗口**：始终显示在屏幕最上层，方便操作
- ✅ **鼠标位置跟随**：在当前鼠标位置周围自动点击
- ✅ **智能防检测**：随机点击位置和间隔，模拟人类操作
- ✅ **一键启停**：简单易用的启动/停止控制
- ✅ **紧急停止**：鼠标移到屏幕左上角自动停止
- ✅ **延迟启动**：点击开始后1秒再执行，给用户准备时间

### 高级功能
- ✅ **操作录制**：记录鼠标点击序列
- ✅ **间隔编辑**：精确调整每个操作的时间间隔
- ✅ **序列回放**：按录制的序列自动执行操作
- ✅ **多模式支持**：单点模式和序列模式切换
- ✅ **实时状态显示**：清晰的运行状态提示

## 🛠️ 技术实现

### 技术栈
- **语言**：Python 3.8+
- **UI框架**：PyQt5
- **鼠标操作**：pyautogui
- **多线程**：threading
- **时间控制**：time

### 防检测机制
- **随机点击位置**：在鼠标位置周围±3px范围内随机
- **随机点击间隔**：基础间隔±20%的随机波动
- **模拟人类移动**：鼠标移动有自然的时间延迟
- **无固定模式**：每次运行的点击模式都不同

## 📦 安装方法

### 方法一：直接运行可执行文件（推荐）
1. 下载 `dist/FloatingClickerPro.exe` 文件
2. 双击运行即可，无需安装Python环境

### 方法二：从源码运行
1. **安装依赖**：
   ```bash
   pip install pyqt5 pyautogui
   ```

2. **运行程序**：
   ```bash
   python floating_clicker.py
   ```

## 🚀 使用指南

### 基本使用
1. **启动程序**：运行 `FloatingClickerPro.exe` 或 `python floating_clicker.py`
2. **准备位置**：将鼠标移动到需要点击的位置
3. **开始点击**：点击窗口中的「Start」按钮
4. **停止点击**：点击「Stop」按钮或将鼠标移到屏幕左上角

### 序列模式使用
1. **切换到序列模式**：点击「模式」下拉菜单选择「序列模式」
2. **录制操作**：点击「开始录制」按钮，然后执行需要的鼠标点击操作
3. **停止录制**：点击「停止录制」按钮
4. **编辑间隔**：在操作列表中右键点击操作，选择「编辑间隔时间」
5. **开始回放**：点击「Start」按钮开始按录制的序列自动执行

## ⚙️ 配置说明

### 基础配置
- **点击间隔**：默认0.15秒，可在代码中修改 `base_interval` 参数
- **随机幅度**：默认±20%，可在代码中修改 `interval_random` 参数
- **点击范围**：默认±3px，可在代码中修改随机数范围

### 高级配置
- **窗口透明度**：默认0.85，可修改 `setWindowOpacity` 值
- **延迟启动时间**：默认1秒，可修改 `time.sleep(1)` 值
- **紧急停止区域**：屏幕左上角，由pyautogui的安全保护机制控制

## 🔒 安全注意事项

- **使用场景**：仅用于合法的自动化操作，请勿用于游戏作弊等违规行为
- **防检测**：虽然实现了基础的防检测机制，但无法保证对所有检测系统有效
- **系统安全**：程序仅模拟鼠标操作，不会修改系统文件或注册表
- **紧急停止**：如遇到异常情况，可通过鼠标移到左上角或强制关闭程序来停止
